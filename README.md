# PWA_Budget

## General Info:
An app that allows you to keep track of your money.
Enter and subtract what you earn and what you spend.
App will work regardless if your offline or online.

## Programs/Languages used:
* Javascript
* JQuery
* Node.js
* Express
* MVC model
* Mongo Database
